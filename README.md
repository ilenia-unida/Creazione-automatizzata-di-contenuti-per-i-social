# Creazione-automatizzata-di-contenuti-per-i-social
✍️ Content Engine AI ➡️ Notion Automatizza la creazione di contenuti. Legge idee per post da Google Sheets, le invia a Gemini (AI) per generare Titolo e Contenuto, e poi carica automaticamente il post in una bozza su Notion. Implementa un controllo condizionale per evitare doppioni
